### Minimum system requirements:

    CPU: 8088 @ 4.77 MHz.
    RAM: 448Kb.
    GRAPHICS: EGA (may not be supported future).
    SOUND: PC Speaker
    
### Recommended system requirements:

    CPU: 80286 @ 8 MHz.
    RAM: 640Kb.
    GRAPHICS: VGA.
    SOUND: OPL2 (AdLib Soundcard/compatible) or MIDI.
